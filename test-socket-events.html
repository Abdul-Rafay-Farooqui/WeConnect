Instructions

  1. Update Socket URL if needed (default: http://localhost:4000)
  2. Click "Connect" button
  3. Wait for "✅ Connected successfully!" message
  4. Click "Join Meeting" button
  5. You should see "✅ meeting:user-joined" event
  6. Open this page in another tab/window with different User ID
  7. Both should see each other join

Expected Results:

  • ✅ Connected successfully!
  • ✅ meeting:user-joined: {"user_id":"test-user-1"}
  • When second user joins: ✅ meeting:user-joined: {"user_id":"test-user-2"}

If Not Working:

  • Check backend console for errors
  • Verify socket URL is correct
  • Make sure backend is running
  • Check CORS settings on backend
  • Verify backend is emitting events correctly
